Design language: **Apple Weather-inspired minimalism** with **Material 3 / Material You** dynamic color.

## Design Principles

1. **Content-first** â€” Weather data is the hero, not chrome/decoration
2. **Glanceable** â€” Key info (temp, condition, high/low) visible instantly
3. **Fluid** â€” Smooth transitions, parallax scrolling, animated icons
