# Screenshots

Place app screenshots here for the README.

## Recommended Screenshots

| Filename | Description |
|---|---|
| `home-light.png` | Home screen (light theme) |
| `home-dark.png` | Home screen (dark theme) |
| `search.png` | City search screen |
| `details.png` | Detailed weather view |
| `widget.png` | Home screen widget |
| `settings.png` | Settings screen |

### Capture Instructions

1. Run the app in Android Studio emulator (Pixel 7, API 34+)
2. Use `Device > Screenshot` or press `Ctrl+S` in the emulator
3. Resize to 1080px wide for consistency
4. Save as PNG with the names above
